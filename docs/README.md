# 2024 Software Design Term Project

## Requirements
### Select a project for refactoring
* The project may be your own project or an open-source project.
* There is no restriction on the programming language or the project size.
* Before refactoring, add unit test cases using an appropriate testing framework.
* The refactoring may be:
  * Micro-level (code-level) refactoring
  * Macro-level (architectural-level) refactoring

### Presentation Slides
* Project introduction
* Introduction to the design of unit test cases
* Explanation of code smells and refactoring
  * Class diagrams
* Execution results and conclusion
